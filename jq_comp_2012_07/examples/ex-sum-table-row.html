This table would be more awesome if it had a 4th column showing the sum of each row.

With jQuery, you must:

  • calculate the sum of each cell in the same row.
  • place each sum in a new column to the right of the existing columns.
  • give each sum a distinctive style, to make it stand out from the other numbers.
